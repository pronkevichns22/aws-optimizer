## 🎯 На самой защите

### Первые 30 секунд (ОЧЕНЬ важны!)
- [ ] Встать прямо, улыбнуться
- [ ] Поздороваться с комиссией
- [ ] Представиться ("Меня зовут..., я защищаю дипломную работу...")
- [ ] Скажите тему (громко и четко)
- [ ] Начните с ПРОБЛЕМЫ, а не с технологий

### Во время презентации
- [ ] Говорите медленнее, чем обычно
- [ ] Делайте паузы после ключевых моментов
- [ ] Не более 2-3 словосочетаний на слайде (не стена текста)
- [ ] Показывайте на экран, когда говорите о слайде
- [ ] Если комиссия задает вопрос - СТОП презентация, ответьте

### На вопросах
- [ ] Слушайте весь вопрос до конца
- [ ] Не перебивайте
- [ ] Если не знаете ответ, скажите честно: "Хороший вопрос, но я этого не рассматривал"
- [ ] Не выглядите обороняющимся - выглядите как эксперт в своей теме
- [ ] Предложите дополнительное объяснение, если нужно

### Завершение
- [ ] "Спасибо за внимание, я готов к вопросам"
- [ ] Дождитесь первого вопроса, НЕ садитесь сразу

## 📋 Финальная проверка текста диплома

```markdown
# Чек-лист перед печатью

## Структура ✓
- [ ] Титульная страница (с подписями)
- [ ] Резюме (на русском)
- [ ] Summary (на английском)
- [ ] Перечень условных обозначений
- [ ] Содержание
- [ ] Введение
- [ ] Глава 1-3
- [ ] Заключение
- [ ] Список источников
- [ ] Приложения (если есть)

## Форматирование
- [ ] Отступы везде одинаковые (1.5 интервал)
- [ ] Шрифт: Times New Roman 12pt или Arial 11pt
- [ ] Нумерация страниц
- [ ] Нумерация рисунков (Рисунок X.X)
- [ ] Нумерация таблиц (Таблица X.X)
- [ ] Все ссылки на источники имеют номера

## Контент
- [ ] Нет опечаток (проверить через Notepad++)
- [ ] Нет пустых абзацев
- [ ] Все рисунки имеют подписи
- [ ] Все таблицы имеют названия
- [ ] Ссылки на рисунки и таблицы в тексте
- [ ] Цитаты оформлены правильно [1]

## Объем
- [ ] Примерно 60+ страниц (как указано в резюме)
- [ ] Введение: 3-5 страниц
- [ ] Глава 1: 10-12 страниц
- [ ] Глава 2: 8-10 страниц
- [ ] Глава 3: 15-20 страниц
- [ ] Заключение: 2-3 страницы
```
