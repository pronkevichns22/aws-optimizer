- ✅ Правильно: "В текущей версии реализован API для получения findings в JSON. 
  Экспорт в PDF, CSV запланирован для версии 2.0. Требуется ~6 часов разработки."

### 2️⃣ "Где Terraform генератор?"

**Что ответить:**
- ✅ Правильно: "Knowledge base содержит AWS CLI примеры. Генерация Terraform 
  кода из рекомендаций - это часть расширения. В дипломе описана архитектура 
  для такой функции, но реализация требует отдельного спринта."

### 3️⃣ "Как работает Prowler интеграция?"

**Что ответить:**
- ✅ Правильно: "PROWLER_SETUP_CHECKLIST.md описывает как настроить Prowler. 
  В первой версии система использует собственный Rules Engine, который проверяет 
  те же CIS Benchmarks. Интеграция с Prowler API планируется."

### 4️⃣ "Почему нет audit logs в базе?"

**Что ответить:**
- ✅ Правильно: "Schema для audit_logs спроектирована в DATABASE_MODELS.md. 
  Логирование в базе требует middleware для всех API endpoints. Это расширение 
  для версии 2.0, так как требует интеграции с каждым handler'ом."

### 5️⃣ "Как пользователь может исправить найденные проблемы?"

**Что ответить:**
- ✅ Правильно: "AI Advisor предоставляет шаги для исправления и примеры AWS CLI команд. 
  Автоматическое исправление через AWS API планируется. Текущий подход - 
  инженер использует рекомендации для ручного исправления или создания Terraform."
